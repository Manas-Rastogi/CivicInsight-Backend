🏛️ CivicInsight Backend: AI-Powered Civic Command CenterCivicInsight is a robust backend engine designed to modernize political constituency management. It automates grievance redressal by using Llama 3 (Local LLM) to classify citizen complaints and assign them to the appropriate officials with strict accountability tracking.🎯 The ProblemPolitical offices in India often struggle with:Fragmented Data: Complaints arrive via WhatsApp, letters, and phone calls with no central record.Lack of Accountability: No way to track if an officer actually resolved an issue.Delayed Response: Manual sorting of thousands of grievances causes weeks of delay.CivicInsight solves this by creating a Digital Command Center for leaders to monitor their constituency in real-time.🏗 System ArchitectureThe backend is built with a Privacy-First mindset, ensuring all AI processing happens on-premise. The WorkflowIntake: Citizen submits a complaint through the mobile/web portal.AI Inference: The backend sends the text to the Ollama (Llama 3) engine.Classification: AI returns the Department (e.g., Sanitation), Priority, and Sentiment.Auto-Assignment: The system calculates the nearest officer with the lowest workload and assigns the ticket.SLA Monitoring: A background cron job tracks the 24/48/72-hour deadlines.Escalation: If unresolved, the ticket is automatically escalated to the Political Admin.🚀 Key FeaturesSmart Dispatcher: Uses a load-balancing algorithm to ensure no officer is overwhelmed.Privacy-First AI: Integration with Ollama ensures no citizen data leaves the local server.SLA & Escalation Engine: Automated triggers for accountability using Spring Scheduler.Role-Based Access Control (RBAC): Secure access levels for Citizens, Officers, and Administrators using JWT.🛠 Tech StackLayerTechnologyLanguageJava 17FrameworkSpring Boot 3.2.xDatabaseMongoDB (Flexible Schema for Grievances)AI EngineOllama (Running Llama 3 8B)SecuritySpring Security + JWTBuild ToolMaven🔌 API Endpoints (v1)MethodEndpointAccessDescriptionPOST/api/v1/complaintsCitizenSubmit new grievance + ImageGET/api/v1/complaints/track/{id}CitizenGet live status and historyGET/api/v1/officer/tasksOfficerView assigned tickets and deadlinesPATCH/api/v1/complaints/{id}/statusOfficerUpdate status with resolution proofGET/api/v1/admin/analyticsAdminFetch heatmap and performance data🤖 Local AI Integration (Llama 3)Instead of expensive cloud APIs, we use Llama 3 via Ollama to maintain data sovereignty.Sample Prompt to LLM:"Act as a Civic Assistant.
